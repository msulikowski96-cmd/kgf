# KGF Taxi App

Aplikacja mobilna dla korporacji taksówkarskiej KGF.

## Uruchamianie aplikacji

### 1. Najprościej: jedno polecenie
Aby uruchomić backend i frontend równolegle:

```bash
npm run dev
```

### 2. Alternatywa lokalna
Do uruchamiania lokalnego (bez zależności od środowiska Replit):

```bash
npm run dev:local
```

### 3. Tryb ręczny
Możesz również uruchomić serwery w osobnych terminalach:

Terminal 1 (Backend):
```bash
npm run server:dev
```

Terminal 2 (Frontend):
```bash
npm run expo:dev
```

## Wymagania

- Node.js (v20+)
- PostgreSQL (dla bazy danych)
